Úvod

Semestrální práce z předmětu Značkovací jazyky se zabývá tvorbou XML dokumentu popisujícího soubor do kterého smyšlená aplikace ukládá svá nastavení. Součástí je také tvorba XSD schéma, XSL transformace do HTML a tato dokumentace. Dokument byl tvořen jako konfigurační soubor počítačové hry.

Metodika

Všechny dokumenty byly vytvořeny v programu NetBeans.

Nejprve byl vytvořen hrubý XML dokument se jmény elementů a atributů. Ten byl následně konverttován do XSD pomocí webové aplikace Freeformater v designu Russian Doll (zachovává stejnou strukturu jako zadnaý XML dokument). Tento XSD dokument byl pak ručně doplněn o restrikce na datové typy a také byla úpravena celková strktuřa dokumenta do finální podoby.

Validita těchto dvou dokumentů byla zkontrolována pomocí webového validátoru . Zároveň bylo zkontrolováno, že XML dokument odpovídá XSD schématu.

XSL transformace do HTML byla psána ručně. Data jsou strukturována do HTML tabulek podle kategorií vyplývajících z designu XSD dokumentu. Do XML dokumentu byl doplněn odkaz na tuto transformaci. Při testování bylo zjištěno že prohlížeče Google Chrome ani Opera neumí s transformacemi pracovat (zobrazí prázdnou stránku). Transformace od HTML neobsahuje všechna data z XML souboru.

Pro XSL transformaci do JSON byl nejprve z XML vygenerován JSON soubor, který byl použit jako základ pro transformaci. Všechny elementy, které obsahovaly pouze jeden atribut a žádnývnořený element, byly přejmenovány podle vzoru element-atribut. Kategorie nastavení vyplícající ze struktury dokumentu byly zabaleny do for-each cyklů, pro zkrácení zápisu adres elementů. Validita JSON souboru vytvořeného transformací byla ověřena na webového validátoru Json Parser Online

Celý projekt byl v průběhu jeho tvorby postupně nahráván (a verzován) na GitHubu.

Popis transformace

JSON

Data jsou transformována do JSON dokumentu. Jsou strukturována stejně jako výchozí XML soubor. Elementy s datovým typem string jsou zapsány v úvozovkách. Číselné datové typy a boolean jsou zapsnány bez úvozovek.

Závěr

Jazyk XML je používán pro strukturované ukládání dat nebo při komunikaci mezi programy. Jeho výhodou je dobrá čitelnost a šikoké možnosti filtrování a omezování dat a datovýh typů. Práce s ním je také snadná, protože k jeho editaci stačí pouze textový editor.
